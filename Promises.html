PROMISES

👉 It is an Object that represent the "eventual" Completion or failure of async talk

👉 Solution to callback hell

👉 let pro = new Promise((resolve,reject)=>{...})

  • 📢Pending
  • 📢Fullfilled/Resolve
  • 📢Rejected

EXAMPLE

        
      function getData(dataId){ 
            return new Promise((resolve,reject)=>{
              setTimeout(()=>{
                console.log(dataId)
            ➡️  resolve("Success")
              },2000)
            })
          }
    

    We want data(1) than(2) than(3)
    

    ➡️ Nested call back (callBack hell)
    ➡️ We pass arrow function as argument (callback)
    

    ➡️ promises.then(res) =>{...} 📢fullfilled
    ➡️ promises.catch(err)=>{...} 📢Rejected
