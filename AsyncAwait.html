ASYNC and AWAIT

👉 always return a promise

👉 Solution to handle async operation

👉 async function myfun(){...} await used inside async
👉 await used inside async . It makes JS wait until promise settle

EXAMPLE

        
      function getData(dataId){ 
            return new Promise((resolve,reject)=>{
              setTimeout(()=>{
                console.log(dataId)
            ➡️  resolve("Success")
              },2000)
            })
          }
    

    ✅async function getAllData(){

        await getData(4)
